## User Stories

### Nutzer zum Verwalten von Themen einer Lehrveranstaltung berechtigen
Als Nutzer in der Rolle 'Admin' möchte ich für eine Lehrveranstaltung festlegen können, ob die Kursteilnehmer Themen anlegen, 
editieren und löschen dürfen, damit die Teilnehmer eigene Themen erfassen können und ich weniger Verwaltungsaufwand habe.

### Mehrere Abgabeartefakte für einzelne Abgaben anlegen
Als Nutzer in der Rolle 'Admin' möchte ich für eine Abgabe beliebig viele Abgabeartefakte festlegen können, damit für die Kurteilnehmer ersichtlich ist, welche Artefakte zu einem Termin einzureichen sind.

### Abgabeartefakte hochladen
Als Nutzer in der Rolle 'Student' möchte ich die notwendigen Abgabeartefakte für eine Abgabe hochladen können,
sodass ich alle Abgabeartefakte für mich oder für meine Gruppe einreichen kann.

### Abgabeartefakte herunterladen
Als Nutzer in der Rolle 'Admin' möchte ich mir die eingereichten Abgabeartefakte für eine Abgabe herunterladen können,
damit ich die Abgabeartefakte ansehen und prüfen kann.
